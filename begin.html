Начало работы

Чтобы начать знакомство с Yandex.Cloud, воспользуйтесь инструкциями:

  • Зарегистрироваться и создать первый платежный аккаунт
  • Начать работу с сервисами:
    • Создайте виртуальную машину в Compute Cloud:
      • Linux
      • Windows
      • Группу виртуальных машин
    • Используйте сервисы для хранения:
      • Разместите файлы в Object Storage.
      • Загрузите Docker-образ в реестр в Container Registry.
    • Настройте сети:
      • Создайте облачную сеть в Virtual Private Cloud.
      • Настройте балансировщик в Load Balancer.
    • Управляйте доступом:
      • Добавьте пользователей и назначьте им роли в IAM.
      • Создайте новый каталог в облаке и предоставьте к нему доступ в Resource Manager.
    • Создайте кластер баз данных:
      • ClickHouse.
      • MongoDB.
      • MySQL.
      • Redis
      • PostgreSQL.
    • Используйте машинное обучение:
      • Преобразуйте текст в речь и наоборот с помощью SpeechKit.
      • Переведите текст в Translate.
      • Анализируйте изображение с помощью компьютерного зрения в Vision.
    • Прочее:
      • Создайте кластер Kubernetes.
      • Настройте обмен сообщениями между приложениями в Message Queue.
      • Визуализируйте данные в DataLens.
